# MovieCatalog - Каталог фильмов

## Описание
Приложение-каталог фильмов, разработанное в рамках лабораторных работ №15-18.
Реализована навигация, работа с ViewModel, StateFlow и корутинами (async/await, coroutineScope).

## Реализованный функционал
- Список фильмов (HomeScreen)
- Детальная информация о фильме с загрузкой данных (DetailsScreen)
- Экран профиля и настроек
- Параллельная загрузка данных через корутины
- Обработка ошибок и индикаторы прогресса

## Технологии
- Kotlin
- Jetpack Compose
- Navigation Compose
- Coroutines (Flow, StateFlow)
- MVVM Architecture
